How to Use GIT


-- Setting Up Git --

1. Make a new Repo
2. Go to where you want your folder.
	-- cd *name here*
3. Clone your repo
	-- git clone *cloning url*


-- Pushing Files to the Cloud --

1. Tell git to track files
	-- git add *file name*
	-- git add *       -- This will add all files.
2. Commit Changes
	-- git commit -m "Your message goes like this"
3.
	-- git push
